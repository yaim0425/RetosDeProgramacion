# Ejercicios de Retos de Programación

Este repositorio contiene la solución a los ejercicios propuestos en la página [retosdeprogramacion.com](https://retosdeprogramacion.com/).

> ⚠️ Nota: La página no especifica un lenguaje de programación para los retos, por lo tanto, **todas las soluciones aquí estarán escritas en Python**.

## Objetivo

Practicar y reforzar habilidades de programación resolviendo problemas de distinta dificultad, organizados por número y fecha de publicación.

## Estructura

Cada reto estará en su propia carpeta con:

- Código fuente en Python
- Explicación o notas si aplica
- Archivos adicionales para pruebas si es necesario
